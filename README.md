# Liquid Syntax Editor for Braze CRM

## Overview

The Liquid Syntax Editor is a web-based tool designed to help marketers and developers create and edit personalized messages using Liquid syntax for Braze CRM. This editor provides a user-friendly interface for selecting templates, editing content while preserving Liquid syntax, and previewing the results in real-time.

## Features

- **Template Selection**: Choose from pre-defined templates to start your message creation process.
- **Syntax Highlighting**: Real-time highlighting of Liquid syntax for easier editing.
- **Live Preview**: See how your message will look with the Liquid syntax rendered.
- **Dark Mode**: Toggle between light and dark themes for comfortable editing in any environment.
- **Responsive Design**: Works seamlessly on both desktop and mobile devices.
- **Copy and Reset**: Easily copy your edited content or reset to the original template.

## Technology Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui components
- next-themes for theme management

## Getting Started

### Prerequisites

- Node.js (version 14 or later recommended)
- npm or yarn

### Installation

1. Clone the repository:

   ```
   git clone https://github.com/your-username/liquid-syntax-editor.git
   cd liquid-syntax-editor
   ```

2. Install dependencies:

   ```
   npm install
   ```

   or if you're using yarn:

   ```
   yarn install
   ```

3. Start the development server:

   ```
   npm run dev
   ```

   or with yarn:

   ```
   yarn dev
   ```

4. Open your browser and navigate to `http://localhost:5173` (or the port specified in your console).

## Usage

1. Select a template from the dropdown menu.
2. Edit the content in the editor while preserving the Liquid syntax.
3. Preview your changes in real-time in the preview pane.
4. Use the Copy button to copy your edited content to the clipboard.
5. Use the Reset button to revert to the original template content.
6. Toggle between light and dark modes using the theme switch in the top-right corner.
